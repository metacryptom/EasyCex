# EasyCex

Uniform api for centralized exchange

## Exchange Supprted

- Binance
- [OKEX v5](https://www.okx.com/docs-v5/en/)
- Huobi
- GateIO

## API

- Mark Data
  - HTTP
  - WebSocket
- Exchange
  - Spot
- Account Manager
  - Balance
  - Withdraw
  - Sub Account
- Statics
  - Order
  - Trade
  - Deposit & Withdraw
  - Reporter
